# TELEGRAM BOT ДЛЯ СБОРА СООБЩЕНИЙ ОТ ПОЛЬЗОВАТЕЛЕЙ В ЧАТЕ

## 1. Запуск проекта

### Склонируйте репозиторий с помощью Git:

```bash
git clone https://github.com/ScriptOtter/telegram_provenfood_bot.git
```

### Настройка .env

Создайте файл .env в корне проекта и заполните его следующими переменными:

```text
NODE_ENV="production"

TELEGRAM_FULL_TOKEN=

GEMINI_API_KEY=

```

### Запуск

```bash
npm run start:dev
```
